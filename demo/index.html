Demo jquery.slide_under.js

  • Achieve the affect of one element sliding under another and disappearing.
  • Does not tweak the dimensions of the underlayer like slideUp() and slideDown().

Homepage Download

Demo Instructions

  1. Click the show link under Example 1a; then click hide.

Example 1a

Instantiate with top element selector as string argument and use explicit form of the hide/show to trigger the slide.

Show Hide

Html

<a href="#" id="show-1a">Show</a>
<a href="#" id="hide-1a">Hide</a>
<img src="images/eiffel-tower.jpg" id="bottom-1a"/>
<div id="top-1a">Overlayer</div>
  • Notice the underlayer comes before the overlayer in the DOM. Do this and you won't have to tweak with z-index.

Javascript

$("#bottom-1a").slideUnder("#top-1a");
$('#show-1a').click(function () {
  $("#bottom-1a").slideUnder('show');
  return false;
});
$('#hide-1a').click(function () {
  $("#bottom-1a").slideUnder('hide');
  return false;
});

Example 1b

Rewrite the Javascript to show how you can access the SlideUnder object directly and trigger it's methods for exactly the same functionality as in 1a.

$("#bottom-1a").slideUnder("#top-1a");
var obj = $("#bottom-1a").data('slideUnder');

$('#show-1a').click(function () {
  obj.show();
  return false;
});
$('#hide-1a').click(function () {
  obj.hide();
  return false;
});

Example Two

Use the toggle option and allow plugin to handle hide/show methods automatically.

Top Layer

Markup

<!-- This element is used to toggle display-->
<div class="toggle">Show/Hide</div> 

<!-- The underlayer element should come first and will appear to slide out from
under the element that comes second. -->
<img src="images/eiffel-tower.jpg" class="bottom-layer"/>
<div class="top-layer">Overlayer</div>

Javascript

$(".bottom-layer").slideUnder({
  "over": ".over-layer",
  "toggle": ".toggle"
});
