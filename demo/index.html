Demo jquery.slide_under.js

  • Three components: wrapper, underlayer, toggle
  • When tapped, the toggle will cause the underlayer to slide out from under the wrapper in a direction.

Example One

Instantiate with selector as argument and use explicit form of the hide/show to trigger the slide.

Show Hide
Overlayer

Html

<a href="#" id="a-show">Show</a>
<a href="#" id="a-hide">Hide</a>
<img src="images/eiffel-tower.jpg" id="a-bottom-layer"/>
<div id="a-top-layer">Overlayer</div>

Javascript

$("#a-bottom-layer").slideUnder("#a-top-layer");
$('#a-show').click(function () {
  $("#a-bottom-layer").slideUnder('show');
  return false;
})
$('#a-hide').click(function () {
  $("#a-bottom-layer").slideUnder('hide');
  return false;
})

Example Two

Use the toggle option and allow plugin to handle hide/show methods automatically.

Markup

<!-- This element is used to toggle display-->
<div class="toggle">Show/Hide</div> 

<!-- The underlayer element should come first and will appear to slide out from
under the element that comes second. -->
<img src="images/eiffel-tower.jpg" class="bottom-layer"/>
<div class="top-layer">Overlayer</div>

Javascript

$(".bottom-layer").slideUnder({
  "over": ".over-layer",
  "toggle": ".toggle"
});
