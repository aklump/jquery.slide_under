Demo jquery.slide_under.js

  • Produces the affect of one element sliding under another element and disappearing.
  • Does not tweak the dimensions of the underlayer like slideUp() and slideDown().
  • Multiple speed algorithms for different effects with adjustable rates.

Homepage Download

Demo Instructions

  1. Click the toggle link; click it again. The image is not destorted during the transition.
  2. Click the show link; then click hide. Notice the overlayer color change; this is due to helper classes slide-under-showing and slide-under-hiding.
  3. Compare speed algorithms constant with absolute.
  4. View more demos.

Html

<a href="#" id="toggle-1a">Toggle
<a href="#" id="show-1a">Show</a>
<a href="#" id="hide-1a">Hide</a>
<img src="images/eiffel-tower.jpg" id="bottom-1a"/>
<div id="top-1a">Overlayer</div>
  • Notice the underlayer comes before the overlayer in the DOM. Do this and you won't have to tweak with z-index.

Javascript

$(window).load(function(){
  $("#bottom-1a").slideUnder("#top-1a");
  $('#show-1a').click(function () {
    $("#bottom-1a").slideUnder('show');
    return false;
  });
  $('#hide-1a').click(function () {
    $("#bottom-1a").slideUnder('hide');
    return false;
  });
  $('#toggle-1a').click(function () {
    $("#bottom-1a").slideUnder("toggle");
  });
});

If your layers contain images, then you should instantiate this plugin inside of $(window).load().
